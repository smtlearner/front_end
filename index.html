Git Hub Manual

First Step : Create a account at https://github.com
Second Step : Create a Repository
at git hub website :
-Click plus sign + .
-Click New Repository
-Give Repository Name
-Check public or private
-Finish Repo creation at Git Hub -Copy the repo HTTP link

To Clone Git Repository from Github Server to local computer

at window cmd : git clone Git Repo's HTTP link

Open VScode IDE

- Open Folder
- Open the path of git clone repo in local computer
- Create a new html file as index.html

File Upload from local Computer to Github Server

at window
at cmd > git add index.html

To make who upload this file - log message

at cmd > git config --global user.email "smtlearner@gmail.com"
at cmd > git config --global user.name "smtlearner"
at cmd > git commit -m "add file"

Finialize Step : To upload the file onto the server

at cmd > git push
- There will be pop up authentication alert box
- Choose web browser authentication
- Type your account password
- After authentication succeed , the upload file will be locate at Your Git Repo
